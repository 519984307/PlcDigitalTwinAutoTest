Blinker

#594


→ Linderonline/FK

Beschreibung

Mit vier Tasten soll sowohl die Frequenz als auch das Tastverhältnis eines Blinklichts eingestellt werden können.

Taster (+) → Frequenz um 10% vergrößern
Taster (-) → Frequenz um 10% verringern




Taster (+) → Ein-Dauer um 10%-Punkte vergrößern
50% → 60% → 70% → 80% → 90%
Taster (-) → Ein-Dauer um 10%-Punkte verringern
50% → 40% → 30% → 20% → 10%




SPS Bezeichnung Kommentar
S1 F- Frequenz reduzieren
S2 F+ Frequenz erhöhen
S3 Tast - Tastverhältnis reduzieren
S4 Tast + Tastverhältnis erhöhen
Reset Reset Alles zurücksetzten
P1 P1 Leuchtmelder
