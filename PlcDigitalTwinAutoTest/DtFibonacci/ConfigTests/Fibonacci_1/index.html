Blinklicht - Fibonacci

#607


→ Linderonline/FK

Beschreibung

Zum Starten der Sequenz muss der Taster S1 gedrückt werden.
Die Lampe soll entsprechend der Fibonacci-Folge blinken (mit einer Auflösung von 100ms).
1,1,2,3,5,8,13,..
D.h.
  • 100ms EIN
  • 100ms AUS
  • 200ms EIN
  • 300ms AUS
  • 500ms EIN
  • 800ms AUS
  • 1,3s EIN
  • fertig

Durch erneutes Drücken von S1 kann der Vorgang wiederholt werden.

SPS Bezeichnung Kommentar
S1 -S1 Taster "Start"
P1 -P1 Meldeleuchte "Störung"
