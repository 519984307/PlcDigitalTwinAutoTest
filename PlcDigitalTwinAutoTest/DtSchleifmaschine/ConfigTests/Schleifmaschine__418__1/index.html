Schleifmaschine


→ Linderonline/FK

Beschreibung

Der Motorschütz wird mit S1 und S2 bedient.

SPS Bezeichnung Kommentar
DigInput.F1 -F1 Thermorelais langsame Drehzahl
DigInput.F2 -F2 Thermorelais schnelle Drehzahl
DigInput.S0 -S0 Taster ( 0 )
DigInput.S1 -S1 Taster ( I )
DigInput.S2 -S2 Taster ( Ⅱ )
DigInput.S3 -S3 Not-Halt
DigInput.S4 -S4 Störung quittieren
DigOutput.P1 -P1 Meldeleuchte "Taster langsam"
DigOutput.P2 -P2 Meldeleuchte "Taster schnell"
DigOutput.P3 -P3 Meldeleuchte "Störung"
DigOutput.Q1 -Q1 Schleifmaschine Langsam
DigOutput.Q2 -Q2 Schleifmaschine Schnell
